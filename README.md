# Quickstart
copy `sample.env` into `.env` and fill in the values.

Install dependancies 
```bash
python3 -m pip install -r requirements.txt
```

Run server locally
```bash
make test
```

Build package for pypi
```
make build
```

Publish package to pypi (see [docs](docs/README.md) for more info)
```
make publish
```
